# algorithm-java
用java描述的数字结构与算法

运行
~~~
//Enter the directory
cd com/assignment

//Compile all files
javac -d . *.java

//Customize commands and allocate memory
alias cryptoGraph="java -Xss64m com/assignment/Main"

//To run the program
cryptoGraph
cryptoGraph -i
cryptoGraph -r assetFile tradeFile
~~~
